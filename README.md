# dropbox-springboot

## Installation : Front-end
    $ git clone git@github.com:niyatpatel23295/dropbox-springboot.git
    $ cd front-end/
    $ npm install
    $ npm run dev

## Installation : Backend
    $ git clone git@github.com:niyatpatel23295/dropbox-springboot.git
    $ cd front-end/
    Import into Eclipse project
    Build with maven
